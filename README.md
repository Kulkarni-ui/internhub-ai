## How It Works
1. User enters:
   - Student skills
   - Internship job description
2. The system:
   - Normalizes and tokenizes text
   - Extracts skills using keyword-based NLP
   - Matches student skills with job requirements
   - Identifies missing skills
   - Calculates ATS confidence score using heuristic logic
   - Generates a resume summary dynamically
3. Results are displayed instantly on the webpage.
